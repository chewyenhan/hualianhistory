⚔️ 大决战模式 · 双人决战

  1. 输入学生名单 → AI生成对战题目
  2. 轮盘随机抽取两名学生 → 各自选择像素勇士角色
  3. 题目出现 + 倒计时 → 双方抢答（按键盘 Q / P 键）
  4. 先抢到者作答 → 答对加分，答错对手得分
  5. 赢者留下继续守擂，输者换下一个学生
  6. 题目用完或只剩一人 → 显示排行榜

🏰 大决战模式 · 组别对决

  1. 设定组别和每组玩家 → AI生成对战题目
  2. 轮盘抽取两组 → 各自选玩家代表 → 选角色
  3. 赢者组守擂（同一玩家继续），输者组换人
  4. 组内所有玩家被淘汰 → 该组退出战场
  5. 剩唯一组别 或 题目用完 → 按组别总分排名

✨ 角色技能说明

💥 大招（主动技能）

连胜3场自动触发技能轮盘。从攻击/防御/反弹/毒雾中随机抽取大招，可能一击反杀对手！

✨ 被动技能（自动触发）

每个像素勇士有独特被动：如狂战士残血伤害翻倍、圣骑士可格挡致死攻击、法师限时答对三题、忍者可额外偷取对手分数等。

  • ⚔️ 狂战士：血量越低伤害越高 | 🛡️ 圣骑士：受致命伤时概率格挡（剩1血）
  • 🧙 法师：三题内答对双倍加分 | 🥷 忍者：抢答成功额外+1分
  • 🏹 弓箭手：对手答错时额外扣血 | 🔮 术士：初次受伤反弹一半伤害
