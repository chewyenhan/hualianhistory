# 🚀 AI 全能竞技场 (AI Quiz Arena) - 教师使用指南

**AI 全能竞技场**是一款专为老师设计的、由 AI 驱动的实时课堂竞技平台。只需粘贴一段课文或资料，AI 即可在几秒钟内为你生成一套高品质的竞赛题，并配合史诗级的视觉与音效，让课堂瞬间变成热血的竞技场。

---

## 🌟 核心亮点
- **零成本出题：** 粘贴资料，AI 自动生成题目、选项、解析及配对 Emoji。
- **动态主题适配：** 界面颜色会根据出题内容自动变换（如：科学类偏蓝，历史类偏古朴）。
- **灵活计分逻辑：** 支持“抢答奖励”与“普通跟进”双重计分机制。
- **决战 BONUS 关：** 最后 3 题自动进入高难度模式，积分翻倍，悬念留到最后一秒。
- **巨幕视觉设计：** 针对教室投影优化，超大号字体确保最后一排学生清晰可见。

---

## 🛠️ 快速上手步骤

### 1. 准备阶段
1. 打开 `index.html`。
2. **输入 API Key：** 在首页输入您的 Gemini API Key（仅保存在您的浏览器本地，安全可靠）。
3. **选择模型：** 点击“检测模型”，建议选择 `gemini-1.5-flash`（速度最快）。

### 2. 配置竞技规则
- **资料内容：** 粘贴你想考查学生的课文、新闻或任何文字资料。
- **题目数量：** 滑动选择 1-30 题。
- **计分设置：** 
  - **普通答题：** 适合所有组别，默认答对 +1，答错不扣分。
  - **抢答奖励：** 针对**手动点击**选出的第一组。默认答对 +2，答错 -1。
- **战队配置：** 设置参赛小组的数量（最多 10 组）和每组人数。

### 3. 开始竞技
- 点击 **“启动 AI 自动出题”**，等待 AI 深度分析完成。
- **随机抽组 vs 抢答：**
  - 使用底部 **“随机抽组”**：抽中的组别按“普通计分”计算。
  - **手动点击** 战队按钮：选中的第一组将被视为“抢答组”，享受/承担更高的奖惩分值。
- **锁定回答：** 各组选定后，点击“揭晓答案”。

---

## 💡 课堂竞技小贴士
1.  **最后三题的翻转：** 开启 BONUS 模式后，最后三题难度会极大增加且分值翻倍。老师可以利用这三题引导学生进行深度思考，这也是落后队伍翻盘的最佳时机。
2.  **随机选人：** 选中一个战队后，点击 **“随机选人”** 按钮。系统会自动报出一个学号（如“请 3 组的 4 号学生回答”），有效防止学霸垄断，让每个学生都保持紧张感。
3.  **知识点复盘：** 每道题揭晓答案后，屏幕会显示**巨型字体的“知识精华总结”**。请务必停顿 30 秒，让全班同学朗读或记录，实现真正的“寓教于乐”。

## 🌐 如何发布到 GitHub (并免费托管)
如果你想通过网址访问，可以按照以下步骤发布：
1. **创建仓库：** 在 GitHub 上新建一个名为 `AI-Quiz` 的公开仓库。
2. **上传文件：** 将本项目文件夹内的所有内容（`index.html` 等）上传到该仓库。
3. **开启 GitHub Pages：**
   - 进入仓库的 `Settings` -> `Pages`。
   - 在 `Build and deployment` 处选择 `main` 分支。
   - 保存后，几分钟内你就会得到一个专属的网站链接。
