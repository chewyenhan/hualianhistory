> 这是一个专为历史教育设计的智能化竞技平台。依托强大的 Gemini AI，将静态的文本瞬间转化为紧张刺激的知识赛场，让学生在博弈中深刻理解历史。

---

## 🌟 核心亮点：双模竞技 & 双重风格

本平台专为**无设备课堂 (No-Device Classroom)** 设计，老师主控，全班互动。

### 1. 两种竞技模式
- **🛡️ 组别竞技模式 (`quiz.html`)**
    - 适合全班分组大战（最高 10 组）。
    - 支持战队自定义、随机抽组、实时锁定答案、动态总分排行榜。
    - 抢答机制：首位抢答组拥有专属奖惩分。
- **🏆 个人挑战模式 (`quiz-solo.html`)**
    - 适合勇士闯关或个人展示。
    - **幸运大转轮**：700px 巨型画布，随机抽取幸运学生。
    - **卡牌选题库**：隐藏式卡牌阵列，学生凭直觉挑战不同分值的神秘题目。
    - **个人英雄榜**：实时金银铜勋章排名。

### 2. 两套视觉风格
- **🟦 未来科技 (Tech Mode)**：深邃的暗蓝色基调，玻璃质感 UI，发光边框，适合大屏幕展示的高级感。
- **🌸 萌趣奶油 (Cute Mode)**：温馨的米黄配色，圆润的卡通设计，柔和的视觉反馈。

### 3. 全方位交互增强
- **音效系统**：内置 BGM 挂件、轮盘转动滴答声、答题对错音效、10秒急促提醒音。
- **TTS 语音系统**：AI 自动朗读题目内容，并在答题结束后朗读“知识精华总结”。
- **文档解析**：直接拖入 PDF 或 Word 课件，AI 即可秒速出题。

## 🛠️ 使用说明

### 第一步：门户入口
访问 `quiz-start.html`，选择您喜欢的视觉风格，点击对应的竞技模式。

### 第二步：智能出题配置
1.  **⚡ 无需配置 Key**：AI 已通过 Cloudflare Worker 内置，打开即用！
2.  **配置题库**：粘贴历史资料，设定总题数。
3.  **难度分配**：手动设置”简单、中等、困难”题目的比例，并设定每种难度的积分分值。
4.  **开始竞技**：AI 将生成符合要求的 JSON 题目数组。

### 第三步：竞技与总结
- **抽签**：使用转轮或组别按钮随机选人。
- **答题**：限时挑战（可自定义 5-30 秒），倒数 10 秒开启音效提醒。
- **复习**：每题结束后显示的“知识精华总结”是复习的关键，系统会自动朗读。
