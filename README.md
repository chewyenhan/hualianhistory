# 🚀 AI 全能竞技场 (AI Quiz Arena) - 教师使用指南

**AI 全能竞技场**是一款专为老师设计的、由 AI 驱动的实时课堂竞技平台。只需粘贴一段课文或资料，AI 即可在几秒钟内为你生成一套高品质的竞赛题，让课堂瞬间变成热血的竞技场。

---

## 💡 核心设计理念：无设备课堂 (No-Device Classroom)
本平台特别针对**学生没有电子设备**的传统课堂环境设计。
- **老师主控：** 老师通过投影仪展示题目和选项。
- **战队竞技：** 学生以小组为单位进行线下讨论、举手抢答或由老师随机抽签。
- **互动回馈：** 老师在屏幕上实时为各组计分，并通过音效和动画营造紧张的竞技氛围。

---

## 🔑 如何申请 API Key (Gemini)
本工具使用 Google 的 Gemini AI。老师需要申请一个免费的 API Key 才能启动出题功能：
1.  访问 [Google AI Studio](https://aistudio.google.com/)。
2.  使用您的 Google 账号登录。
3.  点击左侧的 **"Get API key"**。
4.  点击 **"Create API key in new project"**。
5.  复制生成的字符串（Key），将其粘贴到本平台首页的 API Key 输入框中。
    *   *注：Gemini 1.5 Flash 模型目前提供慷慨的免费配额，非常适合教学使用。*

## 🛠️ 使用说明

### 1. 准备阶段
- **输入 Key：** 粘贴您申请到的 API Key。
- **检测模型：** 点击“检测模型”并选择 `gemini-1.5-flash`。
- **提供资料：** 粘贴你想考查的任何文字资料（课文、新闻、教案等）。
- **设定题数：** 滑动选择 1-30 题。

### 2. 配置规则
- **普通得分：** 适用于随机抽组或跟进组（答对加分，答错不扣分）。
- **抢答得分：** 针对老师**手动点击**选出的第一组。答对有额外加分，答错则会扣分（增加紧张感）。
- **决胜 BONUS 关：** 开启后，最后三题难度自动提升，且积分翻倍。

### 3. 开始游戏
- 老师根据屏幕显示的题目朗读，学生讨论。
- 使用底部的 **“随机抽组”** 或 **“随机选人”** 来决定回答者。
- 揭晓答案后，屏幕会显示**巨型文字总结**，方便全班复习要点。
